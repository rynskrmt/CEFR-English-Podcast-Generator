CEFRレベル調整

A1からC2まで、学習者に合わせた難易度で会話を生成。

単語数コントロール

ポッドキャストの長さを単語数で指定（例: 300語、1000語）。

自然な二人対話

指定した話者名で、まるで本物のポッドキャストのような会話。

高品質な音声合成

OpenAIの最新TTSモデル (tts-1, tts-1-hd) でクリアな音声を生成。

OSS版を使ってみる

簡単セットアップ (3ステップ)

  1. リポジトリをクローン:
    git clone https://github.com/rynskrmt/CEFR-English-Podcast-Generator
    cd CEFR-English-Podcast-Generator
  2. 設定 & 依存関係インストール:
    cp .env.example .env (APIキー等を編集)
    pip install -r requirements.txt
  3. 実行:
    python app.py (Gradioインターフェースが起動します)

詳しくはGitHubリポジトリの README.md をご確認ください。 モデルやボイスの変更は .env ファイルで簡単にカスタマイズ可能です。

さらに手軽に。クラウド版、開発中！

セットアップやAPIキー設定は不要。ブラウザだけで、いつでもどこでもカスタムポッドキャストを作成できるサービスを準備中です。
